## Setup

### External tool for processing files

_No one are mandatory. If they are found, they will be used._

| Functionnality | External tool needed | Download for Windows | Setup on Debian/Ubuntu |
| -------------- | -------------------- | ----------- | ---------------------- |
| Raster image | [ImageMagick](https://imagemagick.org/) | https://imagemagick.org/download/ | `sudo apt install imagemagick` |
| PDF | [XPDF](https://www.xpdfreader.com/) | https://www.xpdfreader.com/download.html | `sudo apt install xpdf` |
